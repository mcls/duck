# Duck

Open a DuckDuckGo search from the command line.

## Install

go get github.com/mcls/duck

## Examples

```
# Default DuckDuckGo search
duck nikola tesla

# The '!bang' commands need to be escaped
duck nikola tesla \!g
duck duck \!gh

# For all bang commands see: https://duckduckgo.com/bang.html
```
